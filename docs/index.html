SumatraPDFControl

SumatraPDFControl is a Windows Forms Control based on SumatraPDF.

It allows you to construct Windows Forms application capable to view and read Portable Document Files (PDF)

  • Download compiled version from NuGet.org

  • View source code licenced under GPLv3 in GitHub

  • Go to Api Documentation to view methods, properties and other documentation.

  • Requires .NET Framework 4.5 or greater

  • This control allows you to open and read PDF files with all features present in great SumatraPDF made by Krzysztof Kowalczyk.

  • It requires a specific compiled SumatraPDF forked code which enables SumatraPDF working in an enhanced plugin mode. Compiled versions of it (32 and 64 bits) are distributed inside the nuget.org package
